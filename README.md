#### How to use

1. Add your project gradle level 

   ```kotlin

   allprojects {
   		repositories {
   			maven { url 'https://jitpack.io' }
   		}
   	}
   ```



2. Add your app level gradle

   ```
   dependencies {
   	       compile 'com.github.HeinXtet:DeevDialog:0.1.2'
   	}

   ```
